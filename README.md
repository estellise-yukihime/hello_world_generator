# **Hello World Generator**

## Table of Contents

- [About](#about)
- [Supported Language](#supported-language)
- [Usage](#usage)

## About

CLI App that will generate hello world from different supported languages.

## Supported Language

- HTML
- Javascript
- C#
- PHP
- Java
- Python
- Go
- Ruby
- Kotlin

## Usage

``` Terminal
1: <path to executable> -h
2: <path to executable> html ~/Downloads/hello.html
```
